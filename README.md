# Coctailz
Coctailz - это приложение - информатор. Помогает узнать о разлчиных коктейлях и других напистках.

## Getting started
```npm install``` <br>
```nodemon server.js```

## MVP Functionality
- поиск напитка по названию
- поиск напитка по ингридиенту
- информация с ингридиентами о напитке
- рандомйзер
